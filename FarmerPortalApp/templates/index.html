Abstract: Blockchain is a method in which a confirmation of a transaction is kept by means of a cryptocurrency. The record is maintained transversely, linking several computers in a peer-to-peer network. Contracts, transactions, and the records of them define the economic system of a country. They set boundaries and provide security to the assets. Considering the features of blockchain such as immutability and maintaining the footage of transaction details, this paper highlights the usage of blockchain technology with a farmer's portal that keeps the footage of selling and buying information of crops.

The proposed solution uses Python as a programming language integrated with the blockchain system that benefits farmers, vendors, and individuals by preserving trade contracts. An interface for farmers is designed using Python programming in combination with blockchain technology to store seller, buyer, selling and buying item details, and total value transacted.
